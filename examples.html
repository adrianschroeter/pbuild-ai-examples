Featured Examples

Practical demonstrations of pbuild-ai capabilities

Basic Package Analysis

Analyze a package source without making any modifications. Perfect for understanding package structure and dependencies.

pbuild-ai
basic analysis

Fix Build Failures

Automatically detect and fix build failures. The AI analyzes build logs and applies appropriate patches.

pbuild-ai --fix
fix intermediate

Update to Upstream Version

Update package to the latest upstream version, rebuild, and fix any compatibility issues automatically.

pbuild-ai --update
update advanced

Update Sources Only

Update package sources to the latest upstream version without running a test build.

pbuild-ai --update-only
update basic

Backport Security Fix

Apply a security fix from upstream using AI-guided patching with custom prompt and --fix mode.

pbuild-ai --prompt "Apply security fix from URL" --fix
security fix intermediate

Simple Modification

Make quick, targeted changes using the --modify flag with a simple text prompt.

pbuild-ai --modify "Add a systemd service file"
basic modify

Cleanup Source

Use --modify to cleanup an existing source according to current packaging rules.

pbuild-ai --modify "Cleanup the source"
basic modify

Deep Analysis Mode

Interactive investigation session inside the build environment. Diagnose complex build failures step-by-step.

pbuild-ai --deep-analyze --interactive --fix
advanced debugging

Generate New Package

Create a complete RPM package from scratch based on a description. AI researches upstream and generates spec files.

pbuild-ai --generate "package ripgrep CLI tool"
generate advanced

Vibe Packaging (Franconian Dialect)

Enable any non-english speaker to vibe packaging. AI understands dialects and finds appropriate software packages.

pbuild-ai --generate "Bagg mer amol a Kardnschbill..." --fix
generate vibe

Update Package with Own Tools

Update a package using custom tool scripts to extend pbuild-ai with package specific automation. Can also be used to gain more permissions, eg. for pushing generated content.

pbuild-ai --update --allow-tool-scripts
update custom advanced

Project Examples

Working with entire projects containing multiple packages

Test Build and Fix Project

Test build an entire project containing multiple packages and automatically fix any build failures across all packages.

pbuild-ai --fix
fix project

Update Sources Only

Update all package sources in a project to their latest upstream versions without running test builds. Fast update for preparation work.

pbuild-ai --update-only
update fast

Update, Build and Fix All

Complete project update workflow: update all sources to the latest upstream, test build everything, and automatically fix any compatibility issues.

pbuild-ai --update
update fix advanced
