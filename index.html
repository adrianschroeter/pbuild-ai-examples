What is pbuild-ai?

pbuild-ai is an AI agent for the openSUSE pbuild tool. It automates package analysis, build fixing, upstream updates, and package generation using AI models provided by an openai api (e.g. ollama).

This site showcases practical examples demonstrating pbuild-ai's capabilities. Each example includes the source package, command used, expected results, and benchmark data. The examples also serve as a functional test suite for pbuild-ai.

pbuild-ai is designed to be used on git-based packages or entire projects. However, it can also be used on legacy package sources of OBS. Working on OBS projects requires some workarounds.

Extendable via skills and scripts. Global, project or package specific.

Secure by default

What is pbuild-ai NOT?

Fast: pbuild-ai often requires many steps and the speed depends heavily on the AI you use. It is actually horribly slow at the moment; its main value is that it can run fully automatically. However, we will improve the speed in the future.

An AI runtime: pbuild-ai depends on an OpenAI-compatible API provider, such as Ollama.

A tool for creating denial of service: pbuild-ai will not push changes or create pull requests by default. However, you could extend it for your project through custom tool scripts. Please use this capability responsibly.

Complete: We have many skills, but there's certainly room for improvement. Many language-specific skills are not yet fully optimized for pbuild-ai.

⚠️ Early Stage Project

pbuild-ai is still in its early stages. This page documents the current state. The results shown in the examples are not claimed to be correct or complete.

However, all provided examples have produced at least once a building result under certain conditions.
