# Results Directory

This directory contains the results of pbuild-ai example runs.

## Structure

```
results/
├── example-name/
│   ├── model-key/
│   │   └── YYYYMMDD_HHMMSS/
│   │       ├── benchmark.json
│   │       ├── metadata.json
│   │       ├── output.log
│   │       ├── diff.patch
│   │       ├── build-1.log
│   │       ├── build-1.log.analyze
│   │       ├── build-1.log.diff
│   │       ├── build-2.log
│   │       └── source/ (gitignored)
```

## Note

- The `source/` directories are excluded from git to keep the repository size manageable
- Build logs and analysis files are included
- Results are generated by `scripts/run-example.sh`
- Report files (HTML, JSON) are generated by `scripts/generate-report.sh`
